后台常驻的小工具，把「返回上一级」变成肌肉记忆。

⚡

打开即用

再次打开 App 不会弹出任何界面，直接执行返回上一级。零 onboarding，零配置。

⌨️

系统快捷键

默认 ⌃⌘↑。若快捷键无效，请到系统设置 → 键盘 → 键盘快捷键 → 服务中启用。

🖱

右键服务菜单

选中任意项目 → 右键 → 服务 → 返回上一级，与访达原生操作习惯一致。

🔗

多种调用方式

支持 CLI、URL Scheme finder-go-up://go-up，可接入超级右键等第三方增强工具。

🚀

极速响应

Unix Socket 进程间通信，AppleScript 预编译与 Finder 连接预热，操作几乎无感延迟。

🔒

权限透明

仅需「控制访达」权限以执行目录导航。无网络请求、无数据收集，源码完全开放。

三步即可上手

安装一次，之后随时可用。

  1. 1

    下载并安装

    从 GitHub Releases 下载 DMG，将 Finder-go-up 拖入「应用程序」文件夹。

  2. 2

    授权访达控制

    首次打开 App 时，若系统提示，请允许控制「访达」。之后 App 会在后台保持运行，供服务菜单与快捷键调用。

  3. 3

    选择你喜欢的方式

    打开 App、按 ⌃⌘↑，或使用右键服务菜单——三种方式任选其一，效果相同。

开发者与第三方集成

脚本、自动化工具、右键增强软件均可调用。

命令行
finder-go-up
URL Scheme
open finder-go-up://go-up
直接打开 App
open ~/Applications/Finder-go-up.app

源码安装后 CLI 位于 ~/.local/bin/finder-go-up，也可手动链接到 /usr/local/bin。

超级右键（iRightMouse Pro）配置
  1. 打开 iRightMouse Pro → 偏好设置 → 工具箱 → 打开 App
  2. 选择 ~/Applications/Finder-go-up.app
  3. 名称设为 返回上一级，保存后重启 iRightMouse Pro
