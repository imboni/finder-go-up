Lightweight, always ready

A background utility that makes “go up one level” muscle memory.

⚡

Launch and go

Relaunching the app shows no UI — it immediately navigates up one level. Zero onboarding, zero configuration.

⌨️

System shortcut

Default shortcut: ⌃⌘↑. If it doesn’t work, enable it under System Settings → Keyboard → Keyboard Shortcuts → Services.

🖱

Services menu

Select any item → right-click → Services → Go Up One Level. Feels native to Finder.

🔗

Multiple entry points

CLI, URL Scheme finder-go-up://go-up, and third-party tools like iRightMouse Pro.

🚀

Fast response

Unix socket IPC, precompiled AppleScript, and warmed Finder connections — actions feel nearly instant.

🔒

Transparent permissions

Only needs permission to control Finder for navigation. No network requests, no data collection, fully open source.

Get started in three steps

Install once, use anytime.

  1. 1

    Download and install

    Download the DMG from GitHub Releases and drag Finder-go-up into your Applications folder.

  2. 2

    Grant Finder access

    On first launch, allow the app to control Finder when prompted. It then stays running in the background for Services and shortcuts.

  3. 3

    Pick your preferred method

    Launch the app, press ⌃⌘↑, or use the Services menu — all three do the same thing.

Developer & third-party integration

Call it from scripts, automation tools, or context-menu enhancers.

Command line
finder-go-up
URL Scheme
open finder-go-up://go-up
Launch the app
open ~/Applications/Finder-go-up.app

After a source install, the CLI lives at ~/.local/bin/finder-go-up. You can also symlink it to /usr/local/bin.

iRightMouse Pro setup
  1. Open iRightMouse Pro → Preferences → Toolbox → Open App
  2. Select ~/Applications/Finder-go-up.app
  3. Set the name to Go Up One Level, save, and restart iRightMouse Pro

Requires macOS 13 or later. The installer is about 1 MB, and uninstall is a single command.
