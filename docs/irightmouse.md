# iRightMouse Pro 集成（可选）

finder-go-up 是独立产品，**不依赖 iRightMouse**。若你已在用 iRightMouse Pro，可将其加入工具箱：

1. 运行 `bash scripts/install.sh` 完成安装
2. 打开 **iRightMouse Pro → 偏好设置 → 工具箱**
3. 添加 **打开 App** 类型
4. 名称填写 **finder-go-up**
5. 选择应用 `~/Applications/finder-go-up.app`
6. 启用该项

独立使用方式见 [README](../README.md)。
